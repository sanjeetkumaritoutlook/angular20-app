Angular Interview Questions for 7+ Years Experience – Real Questions from an Indian MNC. One of my connections recently attended an interview for a Senior Angular Developer role (7+ years experience) at an Indian MNC and shared the actual questions she faced.
Sharing this here so it helps others preparing at a similar level.
Here are the questions asked:
Angular and Architecture
• Promise vs Observable
 • Why use Store (NgRx) instead of shared services?
 • switchMap vs concatMap usecase
 • What are Signals? What types of signals do we have?
 • New features introduced from Angular 16 to latest version
 • Interceptors – real use case, are you aware of mocking using interceptor
 • Reusable components – can we access @Input inside constructor or only in ngOnInit?
 • Which server have you used for deployment?
 • Without CI/CD, how would you deploy an Angular app to Azure?
JavaScript Fundamentals
• Closure
• Hoisting
• Callback function
• What is a pure function?
Logical / Coding Questions
• Find missing number from a number array
• Remove duplicate objects from an array of object with contains fields like empId, name, city, state based on combination of empId and name
• Write a function to explain recursion
